# Ros2 quickstart environment 
This projects aims to provides a environment to  let students of the [smr minor](robotminor.nl) to try out and explore the capabilities of ros2. The 
testing environment also includes a small tutorial of how to create a node. And publish and subscripe data from it. The environment is available via a multitude
of platforms and virtualisation technoligies. Here is a table of the which is available and how much support is has.

|Platform|support|
|--------|-------|
| virtualbox vm | supported |
| hetzner vps | supported |
